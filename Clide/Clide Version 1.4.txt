Purpose:
	Opening multiple daily tools each day can get pretty boring if you do
	it enough times. I figured I'd write a bot that could take care of all the
	things that I was too lazy to do. I merged it recently with Sal with is
	a file moving system with easy customization to any python user to
	improve or build upon. Simple yet works.
